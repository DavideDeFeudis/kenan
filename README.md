# Kenan Dinkelmann Dance

Contemporary dance school web app created with React.js, Node.js and MongoDB.
This application uses create-react-app.

[Visit site](https://dev-kenandinkelmanndance.netlify.com/)

## Installation

```bash
npm install
```

## Usage

```bash
npm start
npm run build
```
